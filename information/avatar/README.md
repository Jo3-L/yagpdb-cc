# Options
## Trigger
**Trigger Type:** `Regex`<br>
**Trigger Value:** `\A-(avatar|av|pfp)`

## Restrictions
* No restrictions

# Description
This command allows you to view the avatar of a given user, defaulting to yourself.

## Usage
`-avatar [user]`
